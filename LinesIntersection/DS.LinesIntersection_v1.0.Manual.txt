Описание:
1. Данная программа создаёт группу отрезков с произвольными координатами начала и конца;
2. Выполняется поиск на их взаимное пересечение;
3. В местах пересечек, тот отрезок, угол наколна которого к горизонтальной оси больше разрывается;
4. Разрезанные отрезки помечаются красным цветом;
5. Все результаты выполненных действий записываются в папку "%USERPROFILE%\Desktop\Logs\".


Инструкция по пользованию:
1. Запустить AutoCad;
2. Выполнить загрузку плагина командой "NETLOAD" указав в диалоговом окне "LinesIntersection.dll";
3. Для запуска программы выполнить команду "DS_CreateLines";
4. Зуммировать пространство модели на созданных отрезках.
