Описание:
1. Данная программа выполняет переименование стилей в Civil3D;
2. Все результаты выполненных действий записываются в папку "%USERPROFILE%\Desktop\Logs\".


Инструкция по пользованию:
1. Запустить Civil3D;
2. Выполнить загрузку плагина командой "NETLOAD" указав в диалоговом окне "DS.StylesRename_vX.X.dll";
3. При загрузке плагина он запустится автоматически. Для повторного запуска выполнить команду "DS_Renamestyles";
4. Если требуется замена текста, ввести в поле "Old name" строку для замены в формате "*sometext*", "sometext*", "*sometext", "sometext";
5. Если в поле "New name" ничего не введено, то программа удалит все вхождения поля "Old name" 
из именования стилей;
6. Если в именовании стиля уже содержится значение поля "Input text to add", то программа ничего 
добавлять не будет;
